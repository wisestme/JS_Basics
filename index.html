JavaScript Language Basics

Variables

Variables are fundamental to every programming language. They are like containers used to hold data. Once data is stored in a variable, it can be reused with the variable in it's stead and not have to rewrite the data. This concept helps us maintain a very important programming concept know as DRY

Data types

  • Number
  • String
  • Boolean
  • Undefined
  • Null

JavaScript has dynamic typing. Data types are automatically assigned to variables

Naming Variables

  • Starts with underscore, dollar sign or letter
  • Forbids use of keywords
  • Symbols except dollar and underscore are not allowed in the middle
  • conventions
    • Use meaningful names
    • use camelCase
